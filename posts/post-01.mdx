## Image Example

Here's how you can add an image. Place your image file (e.g., `my-image.png`) inside the `public/static/images/` directory (or create a new subdirectory like `public/static/blog/` if you prefer). Then use the standard Markdown syntax:

![Alt text for the image](/static/images/my-image.png "Optional image title")

Make sure the path starts with `/static/` if you place it in `public/static/`.

## YouTube Embed Example

To embed a YouTube video, go to the video on YouTube, click "Share", then "Embed", and copy the `iframe` code provided. Paste it directly into your MDX file like this:

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
  allowfullscreen>
</iframe>
